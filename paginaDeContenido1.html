¿Qué es HTML? HTML en estos tiempos se utiliza más para la estructuración de una página web, sus siglas en inglés pertenecen a las palabras (HiperText Markup Language) siendo en español ("Lenguaje de marcas de hipertexto"), en si es un lenguaje que puede servir para muchas otras cosas, y no sólo para estructurar documentos, pero para las demás funciones tenemos a CSS y JS, las cuales explicaremos en futuras publicaciones, en fin HTML se usa para establecer la información que es visualizada en el navegador por el usuario, HTML se basa en el uso de etiquetas, ¿Cómo funciona? se crea un archivo de texto plano, en el cual introducimos etiquetas, por ejemplo para iniciar la etiqueta y para cerrar la etiqueta, en este caso estamos indicando que el contenido de la etiqueta anterior es un párrafo, con las etiquetas podemos hacer que el navegador intérprete lo que necesitemos establecer, todas las etiquetas, comienzan con un signo menor que < y terminan con un signo de mayor que> y muchas de estas utilizan la barra inclinada / para indicar que es el cierre de la etiqueta.
